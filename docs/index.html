Audio Resolution Analyzer

About

Audio Resolution Analyzer is a cross platform application that analyzes PCM encoded audio files. It has the ability to determine if an audio file was cheaply converted from one bit-depth to another, among other things. It also has the ability to convert audio files to another resolution.
